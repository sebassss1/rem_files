# com.basis.zeromessenger

Fork of [ZeroMessenger](https://github.com/annulusgames/ZeroMessenger) adapted for the Basis project.

This package provides ZeroMessenger for use inside the Basis Unity workspace. It is intended to be a drop-in package that mirrors the upstream ZeroMessenger functionality while keeping any Basis-specific integration or fixes together in this fork.

## Usage
Usage follows the upstream ZeroMessenger documentation and examples. Look for sample scenes or example scripts bundled with the package (if present). For API details, refer to the [original project](https://github.com/annulusgames/ZeroMessenger)
